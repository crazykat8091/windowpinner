# How to Use Window Pinner

### 1. Run as Administrator
To pin most games or system-level windows, you must right-click your Python terminal or IDE and **Run as Administrator**. If the app says "User Mode (Limited)", some windows may not stay on top.

### 2. Finding Your Window
- The list refreshes automatically based on the **Interval**.
- Use the **Search** bar at the top to filter windows by name.
- Click **Refresh** to manually update the list immediately.

### 3. Pinning & Focus Lock
- **Pin Checkbox:** Check the 'Pin' checkbox next to a window to make it stay on top.
- **Focus Lock (Checkbox):** When enabled, Window Pinner will send heartbeat messages to the pinned window. This prevents games from pausing or muting when you click on a second monitor.

### 4. Sleep Prevention
- Check **Sleep Prevention** in the bottom bar to prevent your monitors from turning off or the computer from sleeping. This is useful during long downloads or rendering sessions.

### 5. Managing Pins
- Pinned windows stay at the top of the list for easy access.
- Use **Unpin All** to quickly restore all windows to their normal behavior.

---
*Tip: If a game still pauses, ensure Focus Lock is checked and the game is running in "Borderless Windowed" mode rather than "Exclusive Fullscreen".*
